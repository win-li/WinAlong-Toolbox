# 更新日志

本项目遵循语义化版本号。

## [0.4.0] - 2026-07-13

### 新增

- 添加 Portainer CE LTS 部署、状态、日志及启停管理。
- 添加 Docker Compose 项目查询。
- 添加 Portainer SSH 隧道访问说明。
- 添加 Portainer 静态安全测试。

### 安全

- Portainer 默认仅绑定 `127.0.0.1:9443`，不公开管理界面。
- 不启用仅供 Edge Agent 使用的 8000 端口。
- 不提供 Portainer 容器或数据卷删除功能。

## [0.3.0] - 2026-07-13

### 新增

- 添加 Docker 状态、版本、容器、镜像与磁盘占用查询。
- 添加基于 Docker 官方 APT 仓库的 Engine 与 Compose 插件安装。
- 添加经 root 权限检查和二次确认的 Docker 服务启停与重启。
- 添加 Docker 模块静态安全测试，阻止引入数据删除操作。

### 安全

- 不开放端口、不创建应用容器、不清理 Docker 数据，也不自动修改用户组。

## [0.2.0] - 2026-07-13

### 新增

- 添加 Ubuntu/Debian 软件包更新检查与常规更新安装。
- 添加 systemd 时间同步状态检查与启用操作。
- 添加可配置大小的 Swap 创建功能，包含重复检测、二次确认和失败清理。
- 所有新增系统写操作均要求 root 权限并记录日志。

## [0.1.0] - 2026-07-12

### 新增

- 初始化可维护的 Bash 项目骨架。
- 添加循环主菜单和系统检查子菜单。
- 添加系统信息、BBR 状态和 Swap 状态只读检查。
- 添加公共函数、统一 UI、默认配置和日志能力。
- 添加幂等友好的安装脚本与保留日志、备份的卸载脚本。
- 添加关键文件、Bash 语法和可选 ShellCheck 烟雾测试。
- 添加 README、架构说明、忽略规则及运行数据占位目录。
- 固定 Shell 等文本文件使用 LF 换行，便于在 Windows 开发、Linux 运行。

### 修复

- 修复通过 `/usr/local/bin/winalong` 软链接启动时无法定位配置文件的问题。
